Select/deselect the push buttons to create a desired input pattern.
Once the pattern is created, submit it using the Submit button.
The Edit text box shows text for user reference.
An output will be given after all 4 cards are inserted.  
'Boost' says that the system passes command for boosting car, right sequence.
'No boost' says that the sequence was incorrect.
'Lock' says that the device is locked after  consecutive failure attempts, with an alert message box.
After lock, the GUI resets for further testing of the system.
